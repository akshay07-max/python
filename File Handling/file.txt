Game-ich shuru kare ustad.

Chicha ku invite bheje, magar unne nai aaya.

Khair, match-ich start kar diye.

Aaj apan Pochinki mein utre.

Gun-aan le liye daba ke.

Clutch karne gaye the ustad.
Aur wahi baigan mein mil gaye.

Auto recall hogaye.

Phir se Pochinki-ich mein utre.

Wapis tapak gaye.

Random logaan ke pairaan pakad ke recall ki bheek maange.

Hau, recall kare unon.

Phir Pochinki-ich utre.

Phir maut hogayi yaaro.

Poora dimag kharab hogaya, gusse mein game-ich band kar diye!
